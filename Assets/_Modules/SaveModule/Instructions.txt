****************Save Module****************
this module can save any type of data,Without checkFile Name,Path,Exists or not

For Load Data
-only thing you have to do Call "SaveDistributorManager.GetSaveData()" method for load data

For Save Data
-if Auto Save bool checked then it saves automatic OnAppQuit and OnAppPaused events
On cases that you Want to save in spesific times you can call "SaveDistributorManager.SaveData()"

thats it you are ready to use congratz.

@@@@@@Important Note@@@@@@@
*You can use in every life cycle methods

*To use SaveModule properly Add SaveRoot prefab to the scene
